  • User Guide
  • Create a New Event
    • Click on an empty area of the calendar to bring up the events menu. The date and time of the created event will correlate to where the calendar is clicked.
  • Edit or Remove an Event
    • Click on an event tile to open the event's menu. Take care when deleting an event as it cannot be undone.
  • Change Event's Date and time
    • Drag and drop an event tile to set the date and time. Drag the bottom edge of the event tile to make it longer or shorter.
  • Calendar Display mode
    • Use the 'Fullscreen Display Mode' link in conjuction with your browser's fullscreen mode (F11) to display the calendar as a digital sign.
